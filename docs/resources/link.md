# jitsu_link (Resource)

Manages a Jitsu link (connects a stream to a destination). The Jitsu link API has no update endpoint, so any attribute change triggers a destroy+create.

## Example Usage

```hcl
resource "jitsu_link" "stream_to_clickhouse" {
  workspace_id = jitsu_workspace.main.id
  from_id      = jitsu_stream.website.id
  to_id        = jitsu_destination.clickhouse.id

  mode                = "batch"
  data_layout         = "segment-single-table"
  primary_key         = "tenant_id,timestamp,message_id"
  frequency           = 1
  batch_size          = 10000
  deduplicate         = true
  deduplicate_window  = 31
  schema_freeze       = false
  timestamp_column    = "timestamp"
  keep_original_names = false
  functions           = [jitsu_function.inject_tenant_id.id]
}
```

## Schema

### Required

- `workspace_id` (String) - Jitsu workspace ID. Changing this forces a new resource.
- `from_id` (String) - Source stream ID. Changing this forces a new resource.
- `to_id` (String) - Target destination ID. Changing this forces a new resource.

### Optional

- `mode` (String) - Delivery mode (e.g., `batch`, `stream`). Changing this forces a new resource.
- `data_layout` (String) - Data layout (e.g., `segment-single-table`). Changing this forces a new resource.
- `primary_key` (String) - Comma-separated primary key columns. Changing this forces a new resource.
- `frequency` (Number) - Batch frequency in minutes. Changing this forces a new resource.
- `batch_size` (Number) - Maximum batch size. Changing this forces a new resource.
- `deduplicate` (Boolean) - Enable deduplication. Changing this forces a new resource.
- `deduplicate_window` (Number) - Deduplication window in days. Changing this forces a new resource.
- `schema_freeze` (Boolean) - Freeze schema (prevent new columns). Changing this forces a new resource.
- `timestamp_column` (String) - Timestamp column name. Changing this forces a new resource.
- `keep_original_names` (Boolean) - Keep original event property names (no snake_case conversion). Changing this forces a new resource.
- `functions` (List of String) - List of function IDs to apply. Provider adds `udf.` prefix automatically. Changing this forces a new resource.

### Read-Only

- `id` (String) - Link ID (auto-generated by Console).

## Import

Import using `workspace_id/from_id/to_id`:

```shell
terraform import jitsu_link.example <workspace_id>/<from_id>/<to_id>
```
